By moving away from "General AI" usage, this repository establishes a clear boundary between **High-IQ Logical Processing** and **High-EQ Social Interaction**.

## ✨ Key Technical Features

- **Multi-Model Logic Tiering:** Isolates technical debugging (DeepSeek) from physical world-view analysis (Gemini).
- **VLA Architecture Integration:** Leveraging Gemini's Vision-Language-Action capabilities for real-time robotic feedback loops.
- **Empathetic Dialogue Mapping:** Utilizing Pi-AI’s unique EQ-focused transformer architecture for non-linear, supportive conversation.
- **Low-Latency API Workflows:** Streamlined communication protocols for sending sensor data to LLMs and receiving actionable JSON packets.
- **Context Window Optimization:** Managing 1M+ token windows to ensure the AI maintains full awareness of the robot’s long-term hardware history.

## 🛠️ The AI Intelligence Stack

- **Executive Brain:** Google Gemini (High-IQ, Vision, 2M Context).
- **Social Heart:** Inflection Pi-AI (High-EQ, Empathetic Voice).
- **Logic Specialist:** DeepSeek (Precision Coding & Math Reasoning).
- **Inference Model:** OpenAI GPT-4o (General Purpose Logic & Vision).

## 🔌 System Logic & Deployment

### The "Dual-Core" AI approach

To prevent "Personality Blur," the system logic is decoupled into two primary modes:

1. **The Executive Mode (Gemini/DeepSeek):** Focuses on strict code integrity, register mapping for the RP2040, and visual object detection.
2. **The Social Mode (Pi-AI):** Focuses on tone, follow-up questions, and "Rubber Ducking" sessions to help the human developer brainstorm.

### Deployment Workflow

Instead of a single chat window, the AI interacts with the project via:

1. **The Web Interface:** For high-level brainstorming and creative writing.
2. **The API Bridge:** Sending live logs and camera frames to Gemini for immediate hardware-steering commands.
3. **The Voice Channel:** Real-time speech interaction for hands-free robotic control.
