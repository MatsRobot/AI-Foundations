## ✨ Key Technical Features

- **Intelligent Inpainting:** Reconstructing backgrounds behind removed objects with lighting and shadow preservation (Photoroom).
- **Neural Restoration:** Using deep learning to repair scratches and tears in physical photo archives (PhotoBooth.online).
- **Generative Expansion:** Leveraging AI to "outpaint" and extend image borders seamlessly (Pixlr Express).
- **Sub-Pixel Text Recovery:** Dedicated models for sharpening and clarifying blurred or pixelated document text (PicWish).
- **HD Super-Resolution:** Enhancing low-quality assets up to 4x using AI upscaling algorithms (Pixelcut/Pixa).

## 🔌 Selection Logic & Deployment

### The "Functional Tier" Approach

To optimize workflows, tools are categorized by their primary output:

1.  **The Utility Tier (PicWish / Pixa):** Focuses on "fixing" assets—removing backgrounds or clarifying text for technical documentation.
2.  **The Creative Tier (Pixlr / Pho.to):** Focuses on "transforming" assets—adding artistic filters or expanding scenes for social media.
3.  **The Restoration Tier (PhotoBooth):** Focuses on "preserving" assets—digitizing and repairing legacy robotic project archives.

### Deployment Workflow

1.  **Capture:** Take a raw smartphone photo of hardware, documents, or old archives.
2.  **Process:** Pass the image through the relevant AI tool via web interface for instant enhancement.
3.  **Integrate:** Deploy the enhanced, high-resolution asset directly to the GitHub Page or technical README files.
